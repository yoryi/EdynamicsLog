## EDYNAMICSLOG
Prueba Tecnica

## Iniciar Proyecto

```
$ cd ../EDYNAMICSLOG
$ npm install
$ cd ios
$ pod install
```

Correr Proyecto

```
$ npm run ios
$ npm run android

```

- Activación 60FPS
- Vista PhotoQ
